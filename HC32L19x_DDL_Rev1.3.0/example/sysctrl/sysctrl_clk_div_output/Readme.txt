测试环境
================================================================================
测试用板:
---------------------
HC32LFx7(9)x_STK

辅助工具:
---------------------
示波器


辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1） 打开工程编译并下载运行；
2） 连接PB00(RCH输出）、PA01（HCLK输出）和PB01（PCLK输出）到示波器；
3） 重新上电后，按下USER KEY按键，程序开始运行，观察三个IO的时钟输出；
4） 查看示波器显示频率；（RCH = 4M；HCLK = 2M；PCLK = 250K；）
