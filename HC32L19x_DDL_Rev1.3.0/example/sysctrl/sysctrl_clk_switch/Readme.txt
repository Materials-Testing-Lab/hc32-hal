功能描述

本样例主要展示各种时钟源的切换方法
说明：
    无

================================================================================
测试环境
================================================================================
测试用板:
---------------------
HC32LFx7(9)x_STK

辅助工具:
---------------------
示波器


辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1） 打开工程编译并下载运行；
2） 连接PA01(HCLK输出）到示波器；
3） 重新上电后；
4） 依次按下USER KEY键，HCLK切换相应频率输出；
5） 最后一次按下USER KEY键，HCLK切换至RCH 4MHz并输出；
