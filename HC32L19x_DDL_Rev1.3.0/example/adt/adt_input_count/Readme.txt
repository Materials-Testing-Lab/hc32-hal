功能描述
================================================================================
本样例主要展示ADT的单路外部输入脉冲的计数功能
从TRIA（PA03） 输入PWM脉冲，TTimer4的CNTER执行递加计数
说明：
    

================================================================================
测试环境
================================================================================
测试用板: 
---------------------
HC32LFx7(9)x_STK

辅助工具:
---------------------
电源

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）外部输入PWM脉冲信号到PA03(ADT TRIA)
2）打开工程编译并运行。
3）可以调出寄存器窗口查看CNTER往上递加计数
4）TImer4向上计数到达周期值后溢出产生上溢中断，在中断内翻转LED引脚（PD14）电平。
