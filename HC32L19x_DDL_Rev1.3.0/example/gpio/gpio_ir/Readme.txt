功能描述
================================================================================
本样例主要展示GPIO的红外输出功能。
说明：
本样例使用端口复用功能，输出38K红外信号，红外输出极性选择为反向输出。
注：本样例使用定时器模拟UART数据0x55。
（UART数据格式：2400bps，8位数据，1位stop位，无校验位）

================================================================================
测试环境
================================================================================
测试用板:
---------------------
HC32LFx7(9)x_STK

辅助工具:
---------------------
红外发送头和红外接收头

辅助软件:
---------------------
串口调试工具
示波器

================================================================================
使用步骤
================================================================================
1）连接PB09的信号到红外发送头发送端，连接红外接收头接收信号到串口调试工具；
2）打开工程并重新编译；
3）启动IDE的下载和调试功能；
4）将程序下载至Demo板；
5）重新上电，串口调试工具持续收到0x55数据；
   示波器探头连接PB09可查看红外输出信号；
