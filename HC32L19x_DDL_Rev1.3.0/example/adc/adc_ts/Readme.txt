功能描述
================================================================================
本样例主要展示输出片上温度传感器的温度值。

说明：
本样例使用定时器定时启动ADC单次转化，在ADC中断中读取转换结果。当总采样次数完成
后，后台程序对转换结果做平均处理后输出当前温度值。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
HC32LFx7(9)x_STK

辅助工具:
---------------------
无

辅助软件:
---------------------
无。

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能；
3）读取f32TemperValue的值获取当前温度值

================================================================================
注意
================================================================================
本样例使用ADC_CONV_TIMES设置ADC转换次数。可通过修改REF_VOL_SELECT宏定义切换参考
电压值。
